# PistasPadel — Frontend

Aplicació web React + TypeScript + Vite.

## Requisits

- Node.js 20 o superior
- npm 9 o superior

## Instal·lació

```bash
npm install
```

## Desenvolupament

```bash
npm run dev
```

L'aplicació s'obrirà a `http://localhost:5173`.

## Compilació per producció

```bash
npm run build
```

Els fitxers es generaran a la carpeta `dist/`.

## Tests

```bash
npm run test
```
